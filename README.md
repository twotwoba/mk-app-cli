# mk-app-cli

Easily create vite-react web app.

## install

```sh
npm i @ericyuanovo/mk-app-cli -g
```

# Usage

## Create a web app by [vite-react-lite](https://github.com/yokiizx/vite-react-lite)

- Quick Overview

```sh
mk web <project name>
cd <project name>
pnpm i
pnpm run dev
```
